## 記入のヒント
- **種別**: 追加 / 変更 / 矛盾(要修正) / 明確化 / 削除・縮小 / 影響波及（`taxonomy.md` §2）
- **対象箇所**: 章・項番。要件に無ければ「記載なし」。
- **下流根拠**: 「章・項＋要旨」。機密本文の貼り付けは避ける。
- **優先度 / 確信度**: 高 / 中 / 低（`taxonomy.md` §3, §4）
- **採否**: 採用 / 修正採用 / 却下。**却下・修正採用は FB理由を必ず記入**（学習の燃料）。
